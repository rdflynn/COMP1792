Typography Exercises.

Creating good printed material relies on a combination of good typographical skills as well as a good eye for layout - how elements should be "laid out" on the page (i.e. where they appear). In this lab, you will look at the basics of using InDesign, Adobe's DTP (Desktop Publishing) package. Use the resources supplied, and the steps shown in the video, to create a basic layout. Try playing around with the text options as you go through this work to see what makes sense and what looks good and bad - this will be invaluable when helping you understand the difference between good and bad layouts.

Layout.
Matt

Lecture

Layout of elements for print.

Understanding how elements should be laid out on the page is vital - not only is it an aesthetic issue (i.e. making the page look "nice"), it is also a functional issue.
